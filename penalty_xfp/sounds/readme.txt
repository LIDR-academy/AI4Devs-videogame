Sonidos para el juego Penalty Shootout

Estos archivos son placeholders para los efectos de sonido del juego.
Para un mejor resultado, por favor reemplázalos con archivos MP3 reales.

Puedes obtener efectos de sonido gratuitos de:
- https://mixkit.co/free-sound-effects/game/
- https://freesound.org/
- https://soundbible.com/

Nombres de archivos que debes crear:
1. shoot.mp3 - Sonido de disparar el balón
2. goal.mp3 - Celebración de gol
3. save.mp3 - Sonido de atajada o poste
4. whistle.mp3 - Silbato
5. crowd.mp3 - Multitud de fondo (opcional)

Para usar otros formatos como OGG o WAV, actualiza las referencias en script.js.
